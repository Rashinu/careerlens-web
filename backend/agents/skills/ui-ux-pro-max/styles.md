# UI Stilleri — Referans

Bu dosya `SKILL.md` tarafından referans olarak okunuyor. Ajan "stil seçmesi" gerektiğinde buraya bakıyor.

## Popüler stiller

### Glassmorphism
- Arkaplan bulanıklığı (backdrop-blur)
- Yarı saydam kartlar
- Uygun: landing page, portfolio

### Neumorphism
- Yumuşak gölgeler, içine gömülü elementler
- Pastel renk paletleri
- Uygun: kontrol paneli, finans uygulamaları

### Brutalism
- Keskin kenarlar, yüksek kontrast
- Tipografi ağırlıklı
- Uygun: marka siteleri, portfolio

### Bento Grid
- Farklı boyutta kartlar
- Dashboard tarzı
- Uygun: ürün sayfaları, Apple-style landing

### Minimalism
- Bol beyaz alan
- Tek bir vurgu rengi
- Uygun: ciddi iş siteleri, SaaS landing

### Matrix / Hacker
- Monospace font
- Siyah arka plan, neon yeşil/turuncu
- Uygun: teknik içerik, CTF
