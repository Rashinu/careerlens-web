# Renk Paletleri — Referans

Bu dosya `SKILL.md` tarafından referans olarak okunuyor. Ajan "renk paleti seçmesi" gerektiğinde buraya bakıyor.

## Marka / SaaS paletleri

### Terminal Orange (bu sunumun paleti)
- `--bg: #0a0402` · koyu arka plan
- `--orange: #ff8c42` · vurgu
- `--amber: #d97706` · ikincil vurgu
- `--cream: #f5e6d3` · gövde metni
- Uygun: teknik içerik, CTF, hacker/terminal estetiği

### Vercel Black
- `#000000` · arka plan
- `#ffffff` · metin
- `#0070f3` · vurgu mavi
- Uygun: developer tools, dokümantasyon

### Linear Purple
- `#0e0e10` · arka plan
- `#5e6ad2` · vurgu mor
- `#f7f8f8` · metin
- Uygun: issue tracker, proje yönetim

### Stripe Indigo
- `#635bff` · ana marka
- `#0a2540` · koyu yüzey
- `#ffffff` · temiz yüzey
- Uygun: ödeme, fintech

## Nötr paletler

### Notion Minimal
- `#ffffff` + `#191919` + `#787774` aksanlar
- Uygun: bilgi/yazı ağırlıklı ürünler

### Apple Silver
- `#f5f5f7` + `#1d1d1f` + `#0071e3` vurgu
- Uygun: ürün siteleri, e-ticaret

### Terminal Green (klasik matrix)
- `#0d0208` arka plan
- `#00ff41` neon yeşil
- Uygun: CTF, hacker estetiği

## Accent paletleri

### Sunset
- `#ff6b6b` → `#feca57` → `#48dbfb`
- Uygun: yaratıcı portfolio, moodboard

### Pastel Calm
- `#a0c4ff` + `#bdb2ff` + `#ffc6ff`
- Uygun: wellness, meditasyon, eğitim

## Kullanım notu

Palet seçerken:
1. Kullanıcının belirttiği marka tonunu önce sor
2. Backgrounda her zaman kontrast oranı ≥ 4.5:1 olmalı (WCAG AA)
3. Dark mode için alt-palet de öner
